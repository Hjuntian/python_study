
# 1. 一维数组堆叠（隐含加维度）

```python
import numpy as np

a = np.array([1, 2, 3])
b = np.array([4, 5, 6])

# 沿 axis=0 堆叠 → 新数组每个数组变成一行
c0 = np.stack((a, b), axis=0)
print(c0)
print(c0.shape)
````

输出：

```
[[1 2 3]
 [4 5 6]]
(2, 3)
```

注释：

* a, b 是一维数组 (3,)
* stack 会**隐含增加一个维度**，axis=0 → 每个数组作为一行
* shape 变为 (2, 3)

```python
# 沿 axis=1 堆叠 → 新数组每个数组变成一列
c1 = np.stack((a, b), axis=1)
print(c1)
print(c1.shape)
```

输出：

```
[[1 4]
 [2 5]
 [3 6]]
(3, 2)
```

注释：

* axis=1 → 每个原数组作为一列
* stack 内部先增加维度，再沿指定轴堆叠

## 2. 转置（transpose / T）

```python
# 二维数组
d = np.array([[1,2,3],[4,5,6]])
print(d.shape)  # (2,3)

# 转置
e = d.T
print(e)
print(e.shape)
```

输出：

```
[[1 4]
 [2 5]
 [3 6]]
(3,2)
```

注释：

* 二维数组 T = 转置，行 ↔ 列
* 多维数组可以指定轴顺序：np.transpose(array, axes=(1,0,2))，重新排列所有轴

---

## 3. 轴滚动（rollaxis）

```python
# 三维数组示例
f = np.arange(24).reshape(2,3,4)
print(f.shape)  # (2,3,4)

# 把轴 2 移动到最前面
g = np.rollaxis(f, 2, 0)
print(g.shape)
```

输出：

```
(4, 2, 3)
```

注释：

* 原 shape = (2,3,4) → 轴 0=2, 轴1=3, 轴2=4
* rollaxis(f, 2, 0) → 轴 2（最里面的维度）移到最外层（位置0）
* 其它轴顺序保持 → shape 变为 (4,2,3)
* 与 transpose 不同，rollaxis 只移动一个轴，transpose 可以重新排列所有轴

```
